# AI Python Portfolio

안녕하세요! 👋  
이 저장소는 2025년부터 AI 분야 진출을 목표로 한 Python 학습 및 실습 포트폴리오입니다.

## 🛠️ 진행 방법
- 매일 Python 기초부터 AI 응용까지 한 걸음씩 학습합니다.
- 각 학습 내용은 `Day1`, `Day2`, ... 형식으로 파일로 정리합니다.
- 학습한 코드는 Colab에서 작성하고, GitHub에 정리해 업로드합니다.

| ... | (계속 업데이트 예정) | |
